# Support

Local-First RPG is currently a private controlled beta. GitHub help, issues,
and release links work only for authorized repository users; they are not yet a
public support channel.

## Before reporting a problem

1. Confirm the app version in **Settings > About Local-First RPG**.
2. Read [README.md](README.md),
   [release packaging](docs/release-packaging.md), and
   [runtime contracts](docs/runtime-contracts.md).
3. Reproduce with the mock provider when possible to separate app behavior from
   a paid or local model endpoint.
4. Export redacted diagnostics from Settings. Inspect the file before sharing.

Authorized testers can use the repository issue form for normal bugs. Include
the platform, exact steps, expected/actual result, and redacted diagnostics.
Never attach API keys, raw keychain data, an entire runtime database, or private
chat/card content.

For a possible vulnerability, follow [SECURITY.md](SECURITY.md) and do not open
an issue. A verified public help/support destination is required before broad
distribution.
