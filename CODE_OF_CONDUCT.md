# Code of Conduct

## Expected conduct

Contributors must communicate respectfully, critique work rather than people,
protect user privacy, disclose conflicts of interest, and make technical claims
that match retained evidence. Harassment, threats, discrimination, sexualized
attention, deliberate exposure of private data, and bad-faith disruption are
not acceptable.

## Scope

This policy applies in repository discussions, reviews, issue triage, release
coordination, and private project channels when someone is representing the
project.

## Enforcement

Report conduct concerns privately to the repository owner. Do not publish
private evidence. The owner may remove content, limit participation, or revoke
access according to severity and recurrence. If the owner is involved in the
report, use the private channel through which project access was granted and
request an uninvolved reviewer before further distribution.
